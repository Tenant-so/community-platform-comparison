# Contributing

## Corrections

If any feature or pricing data is incorrect, please [open an issue](https://github.com/Tenant-so/community-platform-comparison/issues) with:

1. The platform and feature in question.
2. The correct information.
3. A link to the source (official docs, pricing page, etc.).

## Adding a Platform

To add a new platform to the comparison:

1. Fork the repository.
2. Add the platform to the feature matrix, pricing table, and create a platform profile.
3. Ensure all data is sourced from the platform's official website.
4. Submit a pull request.

## Guidelines

- No affiliate links.
- Data must be verifiable from public sources.
- Keep descriptions factual and concise.
- Pricing should reflect current published rates.
